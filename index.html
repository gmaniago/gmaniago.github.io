SOME THINGS ABOUT ME

It was then that I learned delete didn't simply close an application rather, it erased it.

It was then where I learned more about how a computer works and all the great things you can accomplish with it. Having worked with a fast growing company in my previous career.

I cannot remember how many tickets we submitted to our Software team. In time, through collaboration and team work we were able to amend these growing pains.

It was one thing to be able to point out some things that didn't make sense anymore and one thing to be able to do something about a problem.

We keep moving forward, opening new doors, and doing new things, because we're curious and curiosity keeps leading us down new paths.

SOME OF MY WORK

PaperBack

I like to think of an old school Netflix when describing this project but with books. Using the app, the reader will be able to register and log in to their account. Once logged in they will have the option of either searching for a book or browse books that are sorted through categories. Once the reader chooses one, they will be taken to a separate page that gives them details about the book.

Built With

React.js / Backbone.js / Parse API / Bootstrap / HTML5 / CSS3
